CSS CHEAT SHEET

CSS TYPOGRAPHY

Property Description Syntax
Font Weight Declares how thick or thin should be the characters of a text. font-weight: value;
Font Style Determines the font style in which text will appear. font-style: value;
Line height Declares the vertical spacing between lines of text. font-style: value;

HTML Elements and Structure

Element Description Tag
Ordered List Creates a list of items in sequential order. <ol>
Unordered List It is used to create a list of items in no particular order. <ul>
Strong Highlights important, serious, or urgent text and makes the browser render them bold <strong>
Anchor Anchor element is used to create hyperlinks in an HTML document. <a>
Title Contains a text that defines the title of an HTML document. <title>
